# backend

### initialize
    git clone https://github.com/moeumi/backend
    cd backend
    virtual venv
    venv/Scripts/activate //window
    venv/bin/activate //mac
    pip install flask


### run
    flask run

### tech_stack
- backend: flask
- db: sqlite
- server deploy: AWS lambda, AWS api gateway, AWS S3 by using ZAPPA
